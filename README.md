## API
Make sure you go to [dictionaryapi.com](dictionaryapi.com) and to [pixabay.com](pixabay.com) to get your API key, and put it in the gradle.properties file, otherwise this app will not run.

##Technologies Used: 
- SharePreferences. 
- Fragment.
- AsyncTask. 
- JSON/XML parsing.
